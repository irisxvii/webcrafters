SATURDAY HACKNIGHT

This page explores the details of the official site for Saturday Hacknight, a community-driven hacking event organized by TinkerHub.

  • PURPOSE:
      Provides clear information about upcoming Saturday Hacknight events, including themes, dates, and registration deadlines.
      Provides a dedicated section to showcase interesting projects created during previous Saturday Hacknight events, inspiring future participants.
  • FEATURES:
      A clear and user-friendly navigation system allowing visitors to easily access information about upcoming events, registration details, and resources.
  • STACKS USED:
      HTML, CSS, JavaScript
