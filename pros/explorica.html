EXPLORICA 5.0

This website dives into the official website for Explorica, a flagship event organized by the IEEE chapter of LBSITW.

  • PURPOSE:
      To promote the event showcasing event highlights, speakers and workshops.
      To provide a user-friendly online registration process for attendees.
      Inform participants about important details like schedules, venue information, and code of conduct.
  • FEATURES:
      An FAQ section addressing common inquiries about registration, logistics, and the event itself.
      A dedicated section providing clear directions, maps, accessibility details, and parking information for the event venue.
  • STACKS USED:
      HTML, CSS, JavaScript
